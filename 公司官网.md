## 网站功能概览

### 页面结构
1. **首页** - 展示企业形象、产品亮点、核心优势统计数据和常见问题
2. **产品中心** - 自助洗车机和全自动洗车机分类展示，支持筛选和详情弹窗
3. **关于我们** - 公司介绍、企业文化、发展历程时间线和企业荣誉
4. **服务支持** - 售后服务、技术支持、配件供应、运营指导等服务项目
5. **联系我们** - 联系方式、在线咨询表单

### 技术特性
- **响应式设计**: 完美适配桌面端、平板和移动端
- **交互动画**: 滚动时导航栏背景变化、卡片悬停效果、数字动画、平滑滚动
- **产品筛选**: 支持按类别筛选产品
- **产品详情弹窗**: 点击产品卡片可查看详细参数和特点
- **FAQ手风琴**: 常见问题展开/收起交互
- **联系表单**: 带表单验证和提交成功动画

### 设计风格
- **主色调**: 蓝色系(#0066CC)，代表专业、信任、清洁
- **辅助色**: 橙色(#FF6600)，作为强调色
- **卡片式布局**: 清晰的信息层级
- **现代线性图标**: 使用 Lucide React 图标库

### 项目文件结构
```
src/
├── components/          # 组件目录
│   ├── Header/          # 导航栏
│   ├── Footer/          # 页脚
│   ├── Hero/            # 首页大图区域
│   ├── Features/        # 产品亮点
│   ├── Stats/           # 核心优势统计
│   ├── ProductCard/     # 产品卡片
│   ├── ProductDetail/   # 产品详情弹窗
│   ├── Timeline/        # 发展历程时间线
│   ├── FAQ/             # 常见问题
│   └── ContactForm/     # 联系表单
├── pages/               # 页面目录
├── data/                # Mock数据
└── App.tsx              # 主应用组件
```
